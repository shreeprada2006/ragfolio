## SESGEN (2021 – 2022)

SESGEN is a command-line automation tool built in Python to simplify the process of setting up Amazon SES email identities.

The tool uses AWS services such as SES, Route53, IAM, and Boto3. Its primary goal is to automate the manual steps required to create new domain identities for a hosted zone.

When executed, the tool automatically creates an IAM user, assigns the required permissions, generates an SES identity, and verifies the domain. By automating these steps, SESGEN significantly reduces the time required to configure email sending infrastructure for new domains.

This project demonstrates Maneshwar’s interest in DevOps automation and infrastructure tooling.
